# Tailor Made

## Project Overview
Tailor Made is a comprehensive tailoring application designed to connect customers with tailors, offering transparency and convenience for custom clothing orders.

## Getting Started

To run the project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    # or
    bun install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    # or
    bun dev
    ```

## Technologies
- React
- Vite
- Tailwind CSS
- Shadcn UI
